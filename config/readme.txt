The configuration folder houses everything that you may need to change to adjust the variables on the threat assessment drawing tool.

The gcs.js file is a json var that holds the general characteristics developed/approved by the 1WXG. They might add one or two or
change something so in case that happens it is handy to have them in a easy to reach place to change them quickly.

The network.js file is also a json var that will house all of the ips and folder locations for saving and routing the files.
